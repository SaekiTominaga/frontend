InputSwitch.js Demo

Explicit label

<x-input-switch id="input-switch-1"></x-input-switch> <label for="input-switch-1">switch</label>

* Association is not performed in browsers that do not support formAssociated property. As of March 2022, Firefox 98 and Chrome 99 is supported.

Implicit label

<label><x-input-switch></x-input-switch> switch</label>

* Association is not performed in browsers that do not support formAssociated property. As of March 2022, Firefox 98 and Chrome 99 is supported.

checked attribute

<label><x-input-switch checked=""></x-input-switch> switch</label>

disabled attribute

<label><x-input-switch disabled=""></x-input-switch> switch</label>

storage-key attribute

<label><x-input-switch storage-key="foo"></x-input-switch> switch</label>

Form submission

<label><x-input-switch name="switch1" value="1"></x-input-switch> switch</label>
<label><x-input-switch name="switch2" value="1" checked=""></x-input-switch> switch</label>

* Values are not sent in browsers that do not support ElementInternals. As of March 2023, Firefox 93+ and Chrome 77+ is supported.

Style customization

* CSS custom properties names have changed in the version 2.0.0 update. (Not compatible with version 1 series)
